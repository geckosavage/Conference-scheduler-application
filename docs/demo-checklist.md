# Demo Checklist

## Demo Accounts

- Admin: `admin@conference.local`
- Student: `student@conference.local`
- Password for seeded accounts: `password`

## Suggested Demo Flow

1. Open the home page and show featured conferences and sessions.
2. Visit the conferences page and open a conference schedule.
3. Visit a session detail page and explain room capacity plus speakers.
4. Login as student and register for a session.
5. Open `My Schedule` and show the registered sessions.
6. Logout and login as admin.
7. Open the admin dashboard and show statistics.
8. Create, edit, and delete a conference, room, or speaker.
9. Create a session with overlapping room time to show conflict validation.

## Import Data

After running migrations, import the demo data from `data/seed.sql`.
